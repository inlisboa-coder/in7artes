# IN7Artes v16.0

Versão inicial aplicada ao repositório com conteúdos de Portugal nas 14 áreas.

## Ficheiros principais

- `index.html` — demo autónoma navegável.
- `src/data/in7artes_portugal_v16_data.json` — base de conteúdos Portugal v16.0.
- `src/components/IN7ArtesPortugalV16.tsx` — componente React/TSX para integração.
- `docs/PORTUGAL_v16_14_AREAS.md` — documentação consolidada em Markdown.

## Regra editorial

Cada área usa a estrutura fixa definida:
Artistas, Obras, Fóruns, Mercado, Museus, Redes Sociais, Eventos, Informação complementar, Vídeo Galeria, Galeria de Imagem e Galeria de Redes Sociais.

Mínimo: 10 resultados por subárea.
Objectivo: 20 resultados por subárea.
Portugal e Brasil podem crescer para 50+ resultados.
